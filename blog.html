Question-1: What are the different ways to select an element in the DOM?

The ways to select an element in the Dom are getElementById("id") selects an element by its unique ID, getElementsByClassName("class") selects an element by its class name, getElementsByTagName("tag") selects an element by its Tagname, querySelector("selector") selects the first element that matches a CSS selector, and querySelectorAll("selector") selects all elements that match a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is that innerHTML gets or sets the HTML inside an element, it includes tags. innerText gets only the visible text inside an element, ignoring hidden elements and formatting it like how it appears on the page. textContent gets or sets all text inside an element, including hidden ones.

Question-3: What is event delegation in the DOM?

Event delegation is when instead of adding event listeners to multiple child elements, we add a single event listener to a parent element and use event bubbling to detect which child was clicked. Itâ€™s useful for dynamically added elements and improves performance when handling many elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggered on a child element propagates up to its parent and then further up the DOM tree. If you click a button inside a div, the event first runs on the button, then on the div, then on the body, and so on. This can be useful but sometimes needs to be stopped with event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement("tag"), add them using appendChild(), append(), insertBefore(), and remove them using removeChild() or element.remove().
